# Profile Card Components

A chakra-infused profile card to display user information in a visually appealing way.
Ready to drop into any React app, this card is designed to be both functional and aesthetically pleasing.


<Note title="Inportant">
Make sure to configure the `@rasengan-ui` registry into your `components.json` file.
</Note>

## Profile 01

```bash title="Terminal"
npx shadcn@latest add @rasengan-ui/profile-01
```
