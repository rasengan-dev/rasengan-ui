# Rasengan UI KIT

Rasengan UI offers a curated set of high-quality, ready-to-use UI sections — like hero blocks, pricing sections, testimonials, and more. Each section is crafted with care, powered by Shadcn UI, and easy to summon in any React project.

These React Components can be used with any framework or library, including Rasengan.js, Next.js, Remix, TanStack, React Router, and more. Themeable, responsive, and powered by Shadcn UI — just drop them into your project and build faster.

## Installation

```bash
pnpm install
```

This will install all the dependencies required for the project.

## Run the project

```bash
pnpm run dev
```

This command will open your default browser and navigate to `http://localhost:5320`.

## Usage

When opening the project, you will see a welcome page. And on the bottom left of the page, you will find a button that helps you to open a sidebar menu, where you can find all the components available in the project.

These differents components are available under the `src/components` directory.
